# 小飞英语单词闯关助手 - 产品规格 v2

## 1. Concept & Vision

**定位：** 面向9岁+深圳小学生的英语单词闯关工具，用21天打卡计划帮孩子系统掌握1-3年级核心词汇。

**核心体验：** 每天10分钟，浏览→听写→打卡，简洁有力，不磨叽。答对了有鼓励，答错了温和纠正，没有挫败感，只有成就感。

**目标用户：** 9-12岁小孩，有一定自控力，能理解"坚持21天"这个概念。

**调性：** 简洁有力，不卖萌，不啰嗦。像一个靠谱的学长在带你，不是幼教动画。

## 4. Features & Interactions

### 4.1 解锁规则
- 第1天默认解锁第1、2、3天
- 每天打卡完成后自动解锁下一天（最多同时3天）
- 未解锁日期显示🔒，点击无反应
- 超过3天未打卡，权限不累积，自动清到只保留最近3天

### 4.2 单词浏览（学习模式）
**流程：**
1. 进入今天日期 → 显示当日15词，每页1词
2. 点击🔊播放发音
3. 点击「认识了」→ 标记为认识，下一词
4. 点击「不会」→ 标记为不会，加入复习队列，下一词
5. 全部浏览完 → 进入听写

**单词卡片：**
- 🔊 播放按钮（左上角）
- 英文（大号加粗）
- 中文意思（多行，全部显示）
- 例句（小号斜体）

### 4.3 听写模式
**流程：**
1. 显示中文（遮住英文）
2. 用户输入英文
3. 确认 → 即时反馈
4. 正确/错误动画
5. 显示正确答案（错误时）+ 情绪文案
6. 下一题

**情绪反馈（9岁+简洁有力）：**
- 正确：✅ 对了！ / 💪 不错！ / 🎉 继续保持！/ 🌟 厉害！
- 错误：🤔 没关系，再想想。/ 📝 正确答案：xxx / 🔄 再试一次。
- 完成本天：🎉 第N天完成！明天继续！

### 4.4 打卡系统
- 打卡条件：完成当日15词浏览 + 听写正确率≥60%
- 打卡成功：金色进度更新 + 简短鼓励
- 每天打卡一次，隔天可继续

### 4.5 复习模式
- 错题本里的词专项复习
- 同听写流程，答对从错题本移除
