# 小飞英语单词闯关助手 - 产品规格

## 1. Concept & Vision

**定位：** 一款面向深圳小学1-3年级学生的英语单词闯关工具，简约有趣，让背单词像玩游戏一样上瘾。

**核心体验：** 每天10分钟，完成21天挑战，从"认识它"到"用它造句"，用游戏化的方式建立学习习惯。设计风格偏童趣但不幼稚，色彩明亮但不刺眼——像是给孩子的学习工具，而不是给成人用的app。

**产品调性：** 温暖鼓励型。错了他不会打击你，对了会热情表扬你。像一个永远不会发脾气的耐心老师在陪伴。

## 2. Design Language

### 美学方向
**童趣学院风** — 柔和的彩色背景 + 圆圆的卡片 + 可爱的动画 + 大字体。低龄段孩子用要有趣味，但也要有学习工具的专业感，不能太花哨。

### 调色板
```css
--bg: #FEF9F0;         /* 暖白底 */
--primary: #5B8FF9;    /* 主蓝 */
--success: #5DCA5D;    /* 对了绿 */
--danger: #FF6B6B;     /* 错了红 / 警示 */
--gold: #FFB347;       /* 金黄 / 高亮 */
--purple: #A855F7;     /* 强调紫 */
--text: #2D3748;       /* 深灰文字 */
--muted: #718096;      /* 次要文字 */
--card: #FFFFFF;        /* 卡片白 */
```

### 字体
- 标题：`Fredoka One`（圆润可爱，Google Fonts）
- 单词/句子：`Nunito`（圆润、适合学习，Google Fonts）
- 中文说明：`Noto Sans SC`（思源黑体）
- Fallback: `system-ui, sans-serif`

### 圆角 & 间距
- 卡片圆角：`20px`
- 按钮圆角：`50px`（药丸形）
- 大间距：主题是"呼吸感"，不要拥挤

### 动效哲学
- 单词出现：从下往上弹入（translateY + scale，有点像翻开卡片）
- 对了：绿色边框 + 打勾 + 轻微放大弹跳
- 错了：红色边框 + 轻微抖动（shake）
- 切换页面：淡入淡出
- 进度条：平滑填充，带金色光晕
- 打卡成功：彩纸爆炸庆祝动画

## 4. Features & Interactions

### 4.1 学习模式（默认）
**展示逻辑：** 每天15个单词，分5组，每组3个格子（3×5布局）

**每张单词卡片包含：**
- 英文（大号，加粗）
- 中文释义（中号）
- 小喇叭按钮 → 触发TTS发音
- "认识/不认识" 按钮（影响艾宾浩斯复习）

**操作流程：**
浏览 → 点击"认识"跳过或"不认识"加入复习队列 → 下一词 → 全部完成后统计

### 4.2 听写模式
**展示逻辑：** 屏幕显示中文，用户在输入框输入英文

**交互流程：**
1. 显示中文（例：`苹果`）
2. 用户输入英文（`apple`）
3. 按回车或点"确认"
4. 即时反馈：
   - ✅ 正确：绿色高亮，显示完整单词（英文+中文），进入下一词
   - ❌ 错误：红色抖动，显示正确答案，不记进度（稍后再来）
5. 完成后显示本次成绩（正确率）

### 4.3 复习模式（艾宾浩斯）
根据遗忘曲线，自动安排需要复习的单词：
- 新词第1天学过 → 第2天复习
- 第2天复习 → 第4天再复习
- 第4天复习 → 第7天再复习
- 以此类推

**复习界面同听写模式**，但标注"复习"标签，不影响打卡进度。

### 4.4 打卡系统
- 每天只能打卡一次
- 完成当日学习（或听写正确率≥60%）即可打卡
- 打卡成功：触发庆祝动画 + 进度条更新
- 21天全部完成：解锁"毕业证书"（一张装饰好的完成页面）

### 4.5 我的页面
- 当前打卡天数
- 21天日历视图（完成的格子高亮）
- 错题本（累计记不住的单词列表）
- 数据存储在 localStorage（断网可继续用）

## 6. Technical Approach

### 技术栈
- **纯HTML + CSS + JavaScript**（单文件，无框架）
- 数据存储：**localStorage**（断网可用，无服务端依赖）
- 发音：**Web Speech API**（浏览器内置TTS，中英文支持）

### 数据结构
```javascript
// 词库（全部嵌入HTML）
const WORDS = {
  day1: [{ english: 'I', chinese: '我' }, ...],
  day2: [...],
  ...
};

// 学习进度
const PROGRESS = {
  currentDay: 1,
  checkinDays: [0, 0, 0, ...],  // 21天，0=未完成，1=已完成
  wrongWords: [],                // 错题本
  ebbinghaus: {                  // 艾宾浩斯复习记录
    'I': [lastReviewDate, nextReviewDate, interval]
  },
  checkedInToday: false
};
```

### 艾宾浩斯算法
```javascript
// 新词：当天学完，当天复习一次
// 间隔序列：1天 → 3天 → 7天 → 14天 → 30天
function getNextReviewDate(lastDate, interval) {
  return lastDate + interval * 24*60*60*1000;
}
```
